🎯 발사 벡터 계산 원리

ExampleWorld 발사 로직 시각화

① 마우스 좌표 → 월드 좌표 변환

// 화면 좌표
mouseX = Gdx.input.x + offsetX
mouseY = (screenH - Gdx.input.y) + offsetY

// ↑ Y축 반전: LibGDX는 아래가 0
// ↑ offset: 카메라 이동량 보정

② 탱크 → 마우스 방향 벡터

// 탱크에서 마우스까지 벡터
toX = mouseX - bulletX
toY = mouseY - bulletY

③ 정규화 (길이를 1로)

// 벡터 길이 계산
dist = sqrt(toX² + toY²)

// 길이로 나눠서 크기를 1로
aimX = toX / dist
aimY = toY / dist

④ 불릿에 방향 전달 → 이동

// SuperBullet.update()
x += aimX * speed * delta
y += aimY * speed * delta

// aimX,Y가 1이라 speed만큼만 이동

🔫 Bullet2Twin — 쌍발 발사 원리

fire() 안에서 두 발이 어떻게 만들어지는가

① aim 방향의 수직 벡터

// aim 방향을 90도 회전
verticalVectorX = -aimY
verticalVectorY =  aimX

// → aim과 항상 직각을 이룸

② barrelGap 만큼 좌우 오프셋

// 왼쪽 포신 위치
x - vertical * barrelGap

// 오른쪽 포신 위치
x + vertical * barrelGap

// barrelGap = 30f

③ 두 포신에서 같은 방향으로 발사

// listOf로 두 불릿 동시 반환
return listOf(
  Bullet2Twin(x - vertical * gap, y - vertical * gap, aimX, aimY),
  Bullet2Twin(x + vertical * gap, y + vertical * gap, aimX, aimY)
)
